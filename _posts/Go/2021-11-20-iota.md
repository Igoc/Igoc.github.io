Go 언어의 iota에 대한 정리
{:.notice--primary}

## iota

- 상수에 iota를 사용하면 값을 0부터 순차적으로 할당
- Go 언어에는 열거형이 따로 없기 때문에, iota를 이용하여 상수를 열거형처럼 사용

### 사용법

``` go
package main

import "fmt"

const (
	A = iota
	B
	C
)

const (
	D = iota + 10
	E
	F
)

func main() {
	fmt.Println(A, B, C)
	fmt.Println(D, E, F)
}
```
